# xor
XOR Encryption/Decryption Tool

principle
-----
    1 xor 1 = 0
    1 xor 0 = 1
    0 xor 1 = 1
    0 xor 0 = 0
    so:
      1 xor 1 xor 1 = 1
      1 xor 0 xor 0 = 1
      
usage
-----
    Usage:
	    ./xor --encode            # encode strings which you input
	    ./xor --decode            # decode strings which you input
	    ./xor --help or -h        # show this help infomation
